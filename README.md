# KVNSevBot
> Бот для оповещения о КВН событиях.

---

## Быстрый старт

Для запуска проекта необходимо создать файл `.env` в корне директории со следующим содержанием:

```env
BOT_TOKEN="your_bot_token"
ADMIN_ID="id_main_admin"
PATH_USERS="data/users.db"
```
